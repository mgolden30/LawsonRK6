## RK6-uniform
This repo is asociated with the manuscript "An Explicit Sixth Order Runge-Kutta Method for Simple Lawson Integration" by Matthew Golden. Two new schemes are defined.

1. RK6-uniform. An explicit 8 stage, 6th order RK scheme with equally spaced, ordered stages. 

2. Simple Lawson RK6 (SLRK6). An exponential integrator induced by RK6-uniform that requires a single matrix exponential.

This repo will contain code to reproduce all figures in the paper.
